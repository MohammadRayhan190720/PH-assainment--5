What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, where each element, and piece of text is a node in this tree. The DOM allows to access, manipulate, and update the content, and style of a web page using JavaScript.Mainly DOM creat a Relation Between HTML and Javascript.

How do you select an element from the DOM?
  • By ID: getElementById - Selects a single element with a specific id.

  • By Class Name: getElementsByClassName - Selects all elements with a specific class.

  • By Tag Name: getElementsByTagName - Selects all elements with a specific tag name

  • By CSS Selector (Single Element): querySelector - Selects the first element that matches a CSS selector by id/class/tagName.

  • By CSS Selector (Multiple Elements): querySelectorAll - Selects all elements that match a CSS selector and returns a static NodeList.
How do you manipulate an element's attributes and styles using the DOM?
  • Accessing Elements by getElementById,getElementByTagName,getElementsByClassName
  • Changeing Text useing innerText,innerHTML,innerContent
  • To change an element's attributes, use the setAttribute, getAttribute, or direct property access.
  • Use the style property to manipulate CSS styles.And add class Useing classList
What is event delegation in the context of the DOM, and why is it useful?

Event delegation: add event listeners to each individual child element Instead of you attach a single event listener to a parent element. This parentElement listener can then handle events triggered by its child elements.
